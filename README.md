# database-system

## Startup

```
cd food-donation
```
```
npm install
```
```
cd backend
```
```
npm install
```

note install missing packages for database

```
npm install mongoose pg redis
```


In VS Code, Open 2 terminal
```
cd backend
```

First terminal run
```
ssh -L 5433:localhost:5432 db-dev@35.212.169.134
```

Second terminal run
```
node server.js
```
Project Start
```
npm start
```
