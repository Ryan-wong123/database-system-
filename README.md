## Startup

```
cd food-donation
```
```
npm install
```
```
cd backend
```
```
npm install
```

open first terminal
```
cd food-donation
```
```
cd backend
```
```
ssh -L 5433:localhost:5432 db-dev@35.212.169.134
```

Second terminal run
```
cd food-donation
```
```
cd backend
```
```
node server.js
```

Third terminal run
```
cd food-donation
```
```
npm start
```
